What does it do?
================

This extension provides automatic translation to pages, content and TCA record
fields using `DeepL <https://www.deepl.com/>`__ supported languages.

.. attention::

    As Google Translator is currently possible option, this
    is no longer maintained by this extension and will be dropped in next
    version release.
